## What is Product?

Product is designed around three core principles:

=== "Simplicity"

    Set up in under 5 minutes. No complex configuration required — sensible defaults handle 90% of use cases out of the box.

=== "Reliability"

    Built on battle-tested infrastructure with 99.9% uptime SLA. Automatic failover and replication keep your data safe.

=== "Extensibility"

    A rich plugin ecosystem and open API allow you to integrate with any tool in your stack.
